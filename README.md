# tech-company-website-frontend
Modern tech company landing page featuring responsive design, dark/light theme toggle, dynamic JSON-based content rendering, interactive navigation menu, and contact form. Developed using HTML5, CSS3, and JavaScript.

## Tech Company Website

A modern and responsive tech company website built with HTML, CSS, and JavaScript. This project showcases a professional business landing page featuring dynamic content, a dark/light theme toggle, and a mobile-friendly user experience.

## Overview

The goal of this project was to create a clean, modern, and responsive website for a technology company while practicing front-end development concepts such as responsive layouts, DOM manipulation, JSON data handling, and theme switching.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Mobile navigation menu
* Dark and light mode toggle
* Dynamic Services section powered by JSON data
* Dynamic Development Approach section
* Contact form UI
* Smooth hover effects and animations
* Clean and maintainable code structure

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome
* Google Fonts
